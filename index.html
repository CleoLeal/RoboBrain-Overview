RoboBrain

Um robô inteligente capaz de localizar e manipular objetos conforme solicitado pelo usuário.

O objetivo consiste em criar um robô inteligente capaz de identificar e pegar um objeto escolhido pelo usuário, utilizando imagens captadas por uma câmera para a identificação precisa do objeto..

Tecnologias

Linguagens e bibliotecas

Essas são as linguagens e bibliotecas que utilizamos no proejto.

Python

Utilizamos para desenvolver a interface do projeto, além de gerenciar a comunicação com o Arduino e implementar a lógica de visão computacional.

C++

Foi empregado na programação do Arduino para garantir controle preciso dos componentes físicos.

OpenCV

Utilizado como a base para a visão computacional, permitindo a detecção e análise das imagens capturadas pelo sistema.

YOLOv8

Para o reconhecimento de objetos, treinamos o YOLOv8 com um dataset personalizado, composto pelos objetos escolhidos.
